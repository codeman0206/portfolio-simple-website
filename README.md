# My Portfolio Website

This is a simple portfolio website built using **HTML5** and **CSS3**. It consists of two main pages: **Home** and **Contact**.

The website is designed to present my basic information and provide an easy way for visitors to reach out to me.

### Home Page

* Introduction/About Me section
* Brief overview of my skills or goals
* Clean and simple layout

### Contact Page

* Contact information
* Simple contact form (if included)
* Easy navigation back to Home

## Built With

* HTML5
* CSS3

## Project Structure

portfolio-website/
│── index.html
│── contact.html
│── style.css


## Purpose of This Project

This project helped me practice:

* Creating multi-page websites
* Linking pages using HTML
* Basic CSS styling
* Structuring web pages properly

## How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
